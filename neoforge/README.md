# NeoForge scaffold

This module exists as part of the Architectury project structure, but it is intentionally not wired into the active runtime yet.

Reason:
- the current branch targets Minecraft 1.20.1 first
- the Forge implementation must remain stable
- NeoForge 1.20.1 setup through the current Architectury/Loom stack is not yet stable in this migration branch

The module is kept here so the future NeoForge migration has a fixed place in the project layout.
